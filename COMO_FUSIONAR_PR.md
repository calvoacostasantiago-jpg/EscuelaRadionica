# Cómo Fusionar el Pull Request (PR) - Guía Visual

## El problema
No encuentras el botón para fusionar el PR. Esta guía te muestra exactamente dónde está.

## Pasos para fusionar el PR:

### Paso 1: Ve al Pull Request
Abre este enlace en tu navegador:
```
https://github.com/calvoacostasantiago-jpg/ladingpageescuela/pull/1
```

### Paso 2: Desplázate hacia abajo
Baja hasta el final de la página del PR.

### Paso 3: Busca el botón verde
Verás un botón verde grande que dice **"Merge pull request"** o **"Fusionar solicitud de extracción"** (si GitHub está en español).

El botón está en la parte inferior, después de todos los comentarios y archivos cambiados.

### Paso 4: Haz clic en el botón verde
Haz clic en **"Merge pull request"**

### Paso 5: Confirma la fusión
Aparecerá otro botón que dice **"Confirm merge"** o **"Confirmar fusión"**.
Haz clic en ese botón también.

### Paso 6: Espera 1-2 minutos
GitHub Pages se actualizará automáticamente. Después de 1-2 minutos, tu página funcionará en:
```
https://calvoacostasantiago-jpg.github.io/ladingpageescuela/
```

## ¿No ves el botón?

Si no ves el botón verde "Merge pull request", puede ser por estas razones:

1. **No has iniciado sesión**: Asegúrate de estar conectado a GitHub con tu cuenta
2. **No tienes permisos**: Necesitas ser el dueño del repositorio o tener permisos de escritura
3. **El PR ya está fusionado**: Revisa si en la parte superior dice "Merged" (fusionado)
4. **Hay conflictos**: Si dice "This branch has conflicts", hay un problema que necesita resolverse

## Alternativa: Usar GitHub desde el móvil

Si estás en un móvil:
1. Abre la app de GitHub o el navegador
2. Ve al repositorio
3. Toca "Pull requests"
4. Toca el PR #1
5. Desplázate hasta el final
6. Toca el botón verde "Merge pull request"

## ¿Aún no funciona?

Si después de seguir estos pasos no puedes fusionar el PR, dime:
- ¿Qué mensaje ves en la pantalla?
- ¿Ves algún botón verde?
- ¿Has iniciado sesión en GitHub?
